# План-график (исправленный)
- Учебная направленность: проект используется внутри вуза/образовательной программы.

| № | Этап | Содержание (bullet) | Результат |
| - | --- | --- | --- |
| 1 | Сбор требований | • сценарии пользователя и выбор метрик\n• правила картирования факторов/лимитов\n• макеты потоков выбора метрик | Согласованные сценарии и макеты |
| 2 | Данные и валидация | • загрузка CSV/JSON\n• проверки дат/валют/чисел, лог ошибок\n• примеры входных файлов | Экран/модуль импорта с журналом ошибок |
| 3 | Ядро расчётов и греки | • BS/бином/MC цены\n• Delta/Gamma/Vega/Theta/Rho\n• VaR/ES/LC VaR, сценарный PnL | Библиотека расчётов с тестами |
| 4 | Стресс-сценарии | • каталог исторических/пользовательских шоков\n• пересчёт PnL и подсветка лимитов | Таблицы stress PnL и превышений |
| 5 | Отчёты и экспорт | • таблицы факт vs лимит, stress PnL\n• графики PnL\n• выгрузка CSV/Excel/JSON | Комплект отчётов и графиков |
| 6 | Интеграция/CLI | • CLI для запуска расчётов и экспортов\n• примеры команд и данных | Рабочая точка входа | 
| 7 | Тестирование и документация | • pytest по формулам и краям\n• README/трассируемость\n• фиксация замечаний (hazard/ISS/1000 записей) | Отчёт о тестах, актуальная документация |
