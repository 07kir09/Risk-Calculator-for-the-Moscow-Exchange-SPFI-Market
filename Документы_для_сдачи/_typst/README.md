# Общий Typst-шаблон

Документы устроены единообразно:

- `main.typ` подключает шаблон, конфиг и тело документа;
- `cfg.typ` содержит реквизиты проекта, студента, руководителя и тип документа;
- `body.typ` содержит основной редактируемый текст.

Сборка одного документа:

```bash
typst compile main.typ output.pdf
```

Сборка всех документов при установленном Typst:

```bash
bash Документы_для_сдачи/build_all.sh
```
