# Пояснительная записка и титульный лист

Документ подготовлен в Typst. Основной текст находится в `body.typ`, реквизиты титульных страниц -- в `cfg.typ`, точка сборки -- `main.typ`.

Пояснительная записка описывает фактическую реализацию проекта: backend, frontend, market data, расчетное ядро, workflow и испытания.
