# Документы для сдачи

Комплект редактируемых Typst-документов для командного проекта Risk Calculator for MOEX SPFI.

## Что будем делать

1. Пояснительная записка + титульный лист.
2. Техническое задание:
   - общее;
   - индивидуальное.
3. Руководство оператора или руководство программиста.
4. Программа и методика испытаний:
   - общая;
   - индивидуальная.
5. Текст программы.
6. Ссылка на репозиторий.
7. Описание языка/форматов входных данных.
8. Сведения об отсутствии NDA.
9. Сведения о способе запуска вместо отдельного исполняемого файла.

## Формат подготовки

Документы готовятся в формате Typst:

- `main.typ` - точка входа для сборки PDF;
- `cfg.typ` - реквизиты документа, проекта, студента, руководителя;
- `body.typ` - основной текст документа;
- `_typst/template.typ` - общий шаблон оформления.

Чтобы собрать один документ:

```bash
cd "Документы_для_сдачи/05_Текст_программы"
typst compile main.typ "Текст_программы.pdf"
```

Чтобы собрать все документы:

```bash
bash Документы_для_сдачи/build_all.sh
```
